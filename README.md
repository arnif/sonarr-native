# Sonarr Native

A react native app for [Sonarr](www.sonarr.tv)

## Instructions

### Prerequisites
* [React Native](https://facebook.github.io/react-native/)

### Setup
```
npm install
```
Then just run `react-native run-ios` or `react-native run-android` for the respective platform
